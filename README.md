Quiz Application using Java Swing

A simple desktop-based Quiz Application built using Java Swing.
This application displays multiple-choice questions with a timer, navigation buttons, progress bar, and result evaluation.

Features
Multiple-choice quiz interface
Timer for each question (30 seconds)
Previous and Next navigation buttons
Progress bar to track quiz progress
Score calculation
Result summary with:
Total questions
Correct answers
Wrong answers
Percentage
Performance evaluation
Technologies Used
Java
Swing (GUI)
AWT Event Handling

QuizApplication.java

How It Works
The quiz starts with Question 1.
Each question has 4 options.
User selects one option.
Timer counts down from 30 seconds.
Clicking Next moves to the next question.
Clicking Previous returns to the previous question.
After all questions are completed, the result window is displayed.
